## About The Project

chardb - a place to store your nostalgia. 
chardb was created as a digital place to store pokemon cards as a collection. Forget your excel databases, chardb provides a beautiful UI where you can treasure your collection without getting them out of storage.

Here's why:
* Those pesky excel spreadsheets can't provide the nostalgia you see when looking at an image of a Charmander 
* You can own your database of cards through a UI that is reminiscent of that first episode
* Well, everyone loves Pokemon! :smile:

Once this is live for everyone to use, it will be a place to own your collection. Until then, feel free to clone the repo and contribute!

## Getting Started

You can download/clone the repo and get started in your own code instance.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ashripley/chardb.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your API in `config.js`
   ```js
   const API_KEY = 'ENTER YOUR API';
   ```
